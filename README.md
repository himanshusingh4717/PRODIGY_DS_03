# Task 3: Loan Prediction Using Decision Tree Model
In this task, I developed a Decision Tree model to determine whether a customer is likely to take a loan based on their demographic and financial information.
# 1. Objective
The objective was to create a classification model that predicts customer loan decisions (Yes/No) using a Decision Tree algorithm.
# 2. Data Preprocessing
- Addressed missing values using mean imputation
- Transformed categorical variables through Label Encoding
- Chose important features for training the model
# 3. Model Development
- Divided the dataset into training and testing subsets
- Built and trained a Decision Tree Classifier using Scikit-learn
# 4. Model Evaluation
- Evaluated the model using test data
- Measured prediction accuracy
# 5. Tools and Technologies
- Python
- Pandas
- Scikit-learn
# 6. Key Learnings
- Gained understanding of how Decision Trees function
- Learned the significance of preprocessing data
- Worked with categorical data transformation
- Understood train-test splitting and performance evaluation

This task showed how Decision Tree models can effectively predict customer decisions based on input data.
